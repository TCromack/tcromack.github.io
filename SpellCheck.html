  • Spell Checker

    Checks text for misspelled words

    About this project

    View Project

    For this project I had to create a Spell Checking program using Java. I used a scanner to load in the text to be checked as well as load in the dictionary containing the properly spelled words. I created an Iterable Interface in order to manipulate the dictionary such as add, remove and perform checks as well as lookups on the dictionary later on in the program. I then created an Abstract List to help implement my Interface inside the project. Next I created my Linked List and added functions such as removeFirst, indexOf and contains as necessary functions for the program to work properly. The dictionary look up was easy to create with my framework built out, a quick boolean check to see if the list contained the word and then replace the misspelled word with the correct one. In order to improve lookup times I implemented a Most Recently Used class to put the newest misspelled word at the top of the stack making the program that much more efficient on subsequent misspelled words since users often misspell the same words over and over. This was a fun program to design and helped me learn about scaling out projects but keeping it simple and efficient at the same time.
