    A mock website for a computer parts store

    About this project

    View Project

    This project was my final for my Web Design course. Before taking this course I had very little web development experience and learned a lot by doing this project. I wanted to learn how to use the CSS Flex Box layout. CSS Flex Box makes laying out websites layouts extremely easy with options like flex-direction and flex-wrap to get the visuals you desire. Laying out content within the flexbox is made easy as well with justify-content and align-items. Anyone who has done even a little web design knows that it can be hard to center content vertically and horizontally and flexbox makes it easy by creating a main axis and cross axis to align your content properly.

    I learned how to use Bootstrap to create a responsive website for both desktop and mobile. The benefit of using Bootstrap is the ease of use and how quick it is to make the site responsive by using its built in classes for the navbar and body. The only difficulty I ran into was the framework may have styling that you may not want and you might have to remove it yourself or go into your style sheet and create a work around but the speediness and ease of use out weighs this minor inconvenience. A lot of what I learned in this project I used to create this website and created a good foundation of web development skills I was able to build upon quickly.
