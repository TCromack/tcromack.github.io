    Recent Projects

    Spell Checker

    For this project I had to create a Spell Checking program using Java. I used a scanner to load in the text to be checked as well as load in the dictionary containing the properly spelled words. I created an Iterable Interface in order to manipulate the dictionary such as add, remove and perform checks as well as lookups on the dictionary later on in the program. I then created an Abstract List to help implement my Interface inside the project.

    Fantasy Football Database

    This project was my final for my Database Design and Management course. I set out with the goal of creating a database of Fantasy Football statistics that was able to be searched and sorted through by player name or by player position. The project started using MySQL to create 2 related tables Players and Stats. The Players table contains their name, position, team, and the primary key playerid. The Stats table contains stats such as points, receptions, passing yards, etc. as well as its primary statsid and the foreign key playerid to join the tables later on.

    NHL Stats Analysis

    This project I set out to answer three questions using NHL player statistics. Are younger players penalized more than older players? Surprisingly I found that on average younger players take less penalty minutes on average than older players. I think most people have the belief that on average younger players are less disciplined than older players due to lack of experience but when we take a more in depth look at the statistics we can see that is not the case.
