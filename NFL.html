  • Fantasy Football Website

    A web database of fantasy football stats

    About this project

    View Project

    This project was my final for my Database Design and Management course. I set out with the goal of creating a database of Fantasy Football statistics that was able to be searched and sorted through by player name or by player position. The project started using MySQL to create 2 related tables Players and Stats. The Players table contains their name, position, team, and the primary key playerid. The Stats table contains stats such as points, receptions, passing yards, etc. as well as its primary statsid and the foreign key playerid to join the tables later on.

    At this stage with my tables set and my data ready to be used it was time to move to my ASP.NET MVC Web Application. I created a SQL Server Database within the project then created a new query to get my Tables from the MySQL portion of the project. At this point I could have left the project as is and moved onto generating the models but I wanted the Tables to be Views and not able to be edited by the user so I generated the Views using SQL.

    I then generated Entity Framework Models from the Views and Tables allowing me to add in the Controllers and Views from MVC in order to use within my Web Application. I wanted the Web Application to be sortable by position and player name so I decided to use Javascript and Bootstrap to accomplish this. I created the name filter for the players first and last name by going into the player table and finding the corresponding name searched and hiding the rest. The position sorting was made easy with bootstrap by creating a button and then inside the MVC View creating positional options for the user to sort by using the button.

    The final step was to deploy the Web Application and Database. Using MS SQL I created a Query to bring in my Database for the Web Application to use and linked the Database within the project. I then published the Web Application to my schools website which you can see in action by clicking the View Project button above!
